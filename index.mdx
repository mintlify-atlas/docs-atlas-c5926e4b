Rehike is a custom web frontend server for YouTube that faithfully restores the classic **Hitchhiker** layout — the version of YouTube that existed from 2013 to 2020. It acts as a man-in-the-middle proxy, intercepting YouTube requests and rendering them with authentic archived templates powered by live data from YouTube's InnerTube API.

## What is Rehike?

Rehike is **not a browser extension** — it works above the browser, giving it far greater freedom and portability. Because the entire page is rendered server-side before it reaches the browser, the experience is indistinguishable from the real classic YouTube: no flickering, no uncanny-valley loading states.

## Get Up and Running

<Steps>
  <Step title="Install prerequisites">
    Ensure you have **PHP 8.0+**, **Composer**, and an **Apache** web server (with `mod_rewrite` enabled) available on your system.
  </Step>
  <Step title="Clone and install dependencies">
    Clone the Rehike repository and run `composer install` to pull in Twig, Protocol Buffers, and the DNS library.
  </Step>
  <Step title="Configure Apache">
    Point your Apache virtual host's document root at the Rehike directory. Rehike ships an `.htaccess` that routes all traffic through `index.php`.
  </Step>
  <Step title="Open your browser">
    Navigate to your configured host in any browser. Rehike will proxy YouTube through the classic Hitchhiker layout automatically.
  </Step>
</Steps>

## Key Features

- **Classic Hitchhiker layout** — pixel-accurate recreation of the 2013–2020 YouTube design
- **SPF navigation** — single-page application feel using YouTube's original Structured Page Fragments library
- **Return YouTube Dislike** — built-in integration to restore dislike counts
- **Async PHP engine** — custom Promise/event-loop system for non-blocking InnerTube requests
- **i18n support** — translations available for English, Japanese, Korean, Russian, Portuguese, Polish, Ukrainian, Spanish, and Dutch
- **Built-in debugger** — optional profiler and request logger for development
